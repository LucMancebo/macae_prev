Para garantir a segurança dos dados pessoais e financeiros dos servidores municipais, o MACAEPREV opera em estrita conformidade com a Lei Geral de Proteção de Dados (LGPD).

## O Fluxo de Aceite

Ao acessar o sistema pela primeira vez, ou caso os termos passem por uma atualização legal, uma tela de aceite obrigatória bloqueará o seu acesso ao Painel Operacional.

1. Leia atentamente as responsabilidades do uso dos dados do sistema.
2. Role a página até o fim para habilitar o botão de confirmação.
3. Marque a caixa de seleção afirmando ter lido o documento.
4. Clique em **Concordar e Prosseguir**.

## Auditoria

O seu "Aceite" gera um registro imutável no banco de dados da prefeitura, armazenando a data, hora exata, seu endereço IP e o navegador que você estava utilizando. O repasse não autorizado de CPFs ou informações de margem resultará em bloqueio sumário do operador.
