# Dashboard — adaptação docs-first (histórico rápido)

## O que foi removido

- Seções que eram “conteúdo de processo” (mock/checklist/cobertura funcional/regras de negócio/tópicos de escopo) que não representam telas/entregas do sistema.

## O que foi mantido

- KPIs baseados em dados reais do backend (stats vindas das listagens).
- Alertas/resumo operacional em cards usando badges.
- Tabela de “servidores atualizados recentemente” (dados reais via API).

## O que foi alinhado ao design system

- KPI cards e layout usando tokens CSS (`--bg-card`, `--border-default`, `--shadow-*`, `--radius-*`).
- Tipografia e cores usando aliases do design system.
- Tabela com header/cells coerentes e hover via tokens.
